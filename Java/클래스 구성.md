# Class의 구성

### 클래스의 구성요소

- 클래스 > 멤버, 매소드 > 생성자

- 예제

  **class** Car {           *// 클래스 이름*

  **private** **String** modelName;  *// 필드*

  **private** **int** modelYear;   *// 필드*

  **Car**(**String** *modelName*, **int** *modelYear*) { *// 생성자*

  this**.**modelName **=** modelName;

  this**.**modelYear **=** modelYear;

  }

  **public** **String** **getModel**() { *// 메소드*

  **return** this**.**modelYear \**+\** "년식 " \**+\** this**.**modelName \**+\** " " \**+\** this**.**color;

  }

  }

------

### 필드(field)

- 필드 = 변수

- 예제

  **class** Car {

  **static** **int** modelOutput; *// 클래스 변수*

  **String** modelName;    *// 인스턴스 변수*

  **void** **method**() {

  **int** something **=** 10; *// 지역 변수*

  }

  }

1. **클래스 변수(=공유 변수)** 인스턴스 생성하지 않고 사용 가능하기 때문

- 자동 초기화
- static 키워드 o

1. **인스턴스 변수**

- 자동 초기화
- static 키워드 x

1. **지역 변수**

- method 안에서만 유효

1. **매개 변수**

- method 가 시작하고 끝날때 소멸

  ------

### 매소드(method)

- 반복되는 프로그래밍 피함
- 코드 가독성 up
- 손쉬운 유지보수

------

### 생성자(Constructor)

- 인스턴스 변수를 초기화
- 클래스의 이름과 같게 함
